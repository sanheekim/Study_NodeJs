Package Manager (PM)
- 사용하는 모듈이 많아지면 복잡해짐. 이걸 관리하는 도구.


NPM :
- 대표적인 Package Manager
※ 터미널에 npm install pm2 -g 이비력 :
- npm을 이용해서 pm2를 설치함.
- -g는 이 컴퓨터 어디에서든지 사용할 수 있다는 뜻.

PM2 :
- Advanced, production process manager for Node.js
- 프로그램(process) 혹은 패키지 매니저.
※ pm2 monit 라는 명령어를 친 상태에서 프로그램 강제종료하면
파일을 바로 다시 실행시킴.
